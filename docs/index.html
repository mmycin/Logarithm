Features

Powerful Capabilities

Everything you need to analyze logs efficiently and effectively

AI Assistant

Built-in AI (Logan) analyzes logs, explains errors, and suggests fixes. Supports OpenAI, Anthropic, Gemini, and Groq.

Blazing Fast

Native Rust performance. Handle 10k+ line files instantly with real-time filtering and search.

Advanced Filtering

Filter by level, search with regex, date/time ranges, line numbers, and severity. Fuzzy search included.

Beautiful UI

Modern interface with dark/light themes, syntax highlighting, and smooth animations.

Privacy First

All log processing happens locally. Your data never leaves your machine.

Multi-File Support

Open multiple log files simultaneously with tab-based interface and drag-and-drop.
