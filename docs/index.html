Features

Built for developers

Every feature designed to make log analysis faster, smarter, and less painful.

AI Log Assistant

Ask Logan AI to explain errors, find patterns, and suggest fixes. Works with OpenAI, Anthropic, Gemini, and Groq.

Native Performance

Built with Rust. Handles 10,000+ line files instantly. Real-time filtering with zero lag.

Advanced Filtering

Filter by log level, regex search, date/time ranges, line numbers, and severity. Fuzzy search included.

100% Private

All log processing happens on your machine. Nothing is sent to any server unless you use AI features.

Multi-File Tabs

Open multiple log files simultaneously. Switch between them instantly with a tab-based interface.

Dark & Light Themes

Beautiful dark and light modes. Syntax-highlighted log levels with color-coded badges.

Meet Logan, your log analyst

Right-click any log line and ask Logan to explain it. Add multiple lines as context, mention entire files with @, and get intelligent answers powered by the AI provider of your choice.
